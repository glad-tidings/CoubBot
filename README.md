# CoubBot
CoubBOT is a bot to automate specific tasks using coub accounts.

## Features
| Feature                   | Supported |
| :------------------------ | :-------- |
| Multithreading            | ✅        |
| Auto Claim Task           | ✅        |

## Settings
open project in visual studio and open data.txt add query_id of each account in new line and build or start project
